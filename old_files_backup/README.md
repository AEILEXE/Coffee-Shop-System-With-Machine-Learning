# Coffee-Shop-System-With-Machine-Learning

## Installation

1. Create and activate a virtual environment:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate  # On Windows
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the project:
   ```bash
   python main.py
   ```

## Features

- Point of Sale (POS) system
- Inventory management
- Reports
- User authentication with roles (Owner, Employee)
- Secure password hashing

## Default Users

- Owner: username `owner`, password `OwnerPass123!`
